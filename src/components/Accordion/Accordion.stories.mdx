# Accordion

## Overview

Accordion은 헤더와 관련된 콘텐츠를 표시하거나, 숨기기 위해 클릭할 수 있는 수직으로 쌓인 헤더 목록입니다.

export const Template = (args) => <Accordion {...args} />;
