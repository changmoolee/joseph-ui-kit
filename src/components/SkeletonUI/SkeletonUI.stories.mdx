# SkeletonUI

## Overview

SkeletonUI는 데이터가 로드되기 전에 콘텐츠 미리보기를 표시하여, 사용자가 로드를 기다리게 합니다.

export const Template = (args) => <SkeletonUI {...args} />;

## Note

&bull;&nbsp; SkeletonUI 내부에 넣을 contents를 element 형태로 'children' props에 넣어줍니다.<br/><br/>
&bull;&nbsp; 아래는 다양한 모양을 가진 element들이 'children'으로 들어간 예시입니다.

<Canvas>
