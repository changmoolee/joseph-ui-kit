# Typography

## Overview

IBM Plex를 서체로 사용합니다.

## Note

### Typography 모듈 적용하기

<div style={{ marginLeft: "30px" }}>
(1) Typography 모듈을 Import 합니다.

<div style={{ marginLeft: "30px" }}>
  `ex) import {(UtilityStyles, BodyStyles)} from "../Typography/Typography";`
</div>

(2) 적용하길 원하는 컴포넌트에 코드를 추가합니다.

<div style={{ marginLeft: "30px" }}>`ex) ${BodyStyles.bodyCompact01}`</div>
</div>

### SASS 파일에서 Typography 적용하기

<div style={{ marginLeft: "30px" }}>
(1) joseph-sass 패키지를 npm을 통해 다운 받습니다.

<div style={{ marginLeft: "30px" }}>`npm install joseph-sass`</div>

(2) 적용하길 원하는 sass파일 내에 명령어를 작성합니다.

<div style={{ marginLeft: "30px" }}>`@use "joseph-sass/type"`</div>

(3) 원하는 클래스나 엘리먼트에 명령어로 적용합니다.

<div style={{ marginLeft: "30px" }}>
  `@include type.UtilityStyles(helperText02)`
</div>

</div>
