# FileUploader

## Overview

FileUploader를 사용하면 사용자가 이미지 파일을 특정 위치에 업로드할 수 있습니다.

export const Template = (args) => <FileUploader {...args} />;

## How to use

- 'fileSize' props에는 FileUploader에 업로드할 수 있는 이미지 파일 사이즈 한도를 설정할 수 있습니다. (단위는 메가바이트(MB)입니다.)<br/><br/>
- 'buttonKind, buttonName' 등 props로 자유로운 Button 커스터마이징이 가능합니다.<br/><br/>
- 'onChange' props의 첫번째 인수(event)로 FileUploader에서 변화가 일어날 때 발생하는 이벤트 값을, 두번째 인수(data)로 FileUploader에
  인코딩되어 입력된 파일 string을 가져올 수 있습니다.<br/><br/>
  `ex) onChange={(event, data) => console.log(event,data)}`
