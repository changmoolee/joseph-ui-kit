# DropDown

## Overview

DropDown은 사용자가 하나 또는 여러 옵션을 선택할 수 있는 옵션 목록을 제공합니다. <br/>선택한 옵션은 양식의 값을 나타내거나, 기존 콘텐츠를 필터링하거나, 정렬하는 작업으로 사용할 수 있습니다.

export const Template = (args) => <DropDown {...args} />;

## Note

&bull;&nbsp; 'items' props에는 아래 예시의 코드와 같이 `[{id: "", value: ""}]` 구조를 가진 배열을 넣어줘야 합니다.<br/><br/>
&bull;&nbsp; 'label, warn, width' 등 props로 자유로운 커스터마이징이 가능합니다.<br/><br/>
&bull;&nbsp; 'onChange' props를 통해서 DropDown의 선택된 값을 가져올 수 있습니다.<br/>
