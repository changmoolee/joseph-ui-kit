# TextArea

## Overview

TextArea는 사용자가 키보드를 통해 앱에 텍스트를 입력하기 위한 기본 구성 요소입니다. 사용자가 여러 줄의 텍스트를 입력할 수 있습니다.

export const Template = (args) => <TextArea {...args} />;

## How to use

- 'id, rows, maxLength' 등 props로 textarea 요소의 속성값을 설정합니다.<br/><br/>
- 'width, label, warn' 등 props로 자유로운 TextArea 커스터마이징이 가능합니다.<br /><br />
- 'onChange' props를 통해서 TextArea에 입력된 값을 가져올 수 있습니다.<br/><br/>
  `ex) onChange={(data) => console.log(data)}`
